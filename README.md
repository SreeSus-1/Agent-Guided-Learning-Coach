# Agent-Guided-Learning-Coach
An AI tutor system where agents support different learning functions.


This project is an AI tutoring system built with multiple agents that support different parts of the learning process. The Explainer Agent teaches and simplifies a selected topic in beginner-friendly language, the Quiz Agent generates questions to test understanding of that topic, and the Motivation Agent gives encouragement and study advice based on the learner’s progress. A tracker stores topic-wise progress and self-assessment scores so the system can maintain memory across learning sessions. Together, these agents create a feedback loop where the user learns a concept, checks understanding, receives support, and tracks improvement over time in a Streamlit-based learning workspace.
